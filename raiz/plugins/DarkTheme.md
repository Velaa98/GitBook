## Theme-api
Nos permite añadir un tema algo más oscuro a nuestra página, tiene muchos
[parámetros de configuración](https://www.npmjs.com/package/gitbook-plugin-theme-api?activeTab=readme).

### Configuración
```
{
  "plugins":
    [
    "theme-api"
    ],
  "pluginsConfig":
    {
        "theme-api":
        {
        "theme": "dark"
        }
    }
}
```
