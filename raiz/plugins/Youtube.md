## Youtube
Nos permite insertar videos de youtube en nuestra página de forma muy simple, solo debemos usar dos
etiquetas junto al enlace del video deseado:

```
{% youtube %} https://youtu.be/eKdeaDbNwnw {% endyoutube %}
```

### Configuración
```
{
  "plugins":
    [
    "youtube"
    ]
}
```

### Ejemplo de uso
{% youtube %} https://youtu.be/eKdeaDbNwnw {% endyoutube %}
